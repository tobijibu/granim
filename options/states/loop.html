loop

更新日: 2019.12.19

loop

アニメーションをループするかどうかを指定します。

  • 初期値 : true
  • type : boolean

グラデーションのアニメーションが最後まで到達したら繰り返しますか？


デモ

'no-repeat'ボタンを押すと、ループを止めます。

no-repeat
repeat
JavaScript
var granimInstance = new Granim({
  element: '#granim-canvas',
  opacity: [1, 1],
  states : {
    'default-state': {
      gradients: [
        ['#834D9B', '#D04ED6'],
        ['#1CD8D2', '#93EDC7']
      ],
    },
    'state-no-repeat': {
      gradients: [
        ['#834D9B', '#D04ED6'],
        ['#1CD8D2', '#93EDC7']
      ],
      loop: false,
    }
  }
});

document.querySelector('.default-state').addEventListener('click', function() {
  granimInstance.changeState('default-state');
});

document.querySelector('.state-no-repeat').addEventListener('click', function() {
  granimInstance.changeState('state-no-repeat');
});
